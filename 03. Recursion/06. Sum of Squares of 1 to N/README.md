# Sum of Squares of 1 to N

**Difficulty:** Easy  
**Time Complexity:** O(N)  
**Space Complexity:** O(N)  

---

You're given a positive integer `n`.  
Your task is to calculate the **sum of squares of all integers from 1 to n** using **recursion**.

### Example 1
Input:  
n = 3  

Output:  
14  

**Explanation:**  
1² + 2² + 3² = 1 + 4 + 9 = 14

### Example 2
Input:  
n = 5  

Output:  
55  

**Explanation:**  
1² + 2² + 3² + 4² + 5² = 55

### Your Task
Complete the function `recursiveSumOfSquares()` that takes an integer `n` as an argument and returns the sum of squares of all integers from `1` to `n` using recursion.

### Constraints
- 1 ≤ n ≤ 950  

### Input Format
- The input contains a single integer `n`.

### Output Format
- Output is a single integer representing the sum of squares from `1` to `n`.
