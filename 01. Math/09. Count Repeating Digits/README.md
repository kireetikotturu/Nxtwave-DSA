# Count Repeating Digits

**Difficulty:** Medium  
**Time Complexity:** O(log₁₀N)  
**Space Complexity:** O(1)  

---

You are given an integer `num`.  
Your task is to count the **total number of digits that are repeating** in the number.

### Example 1
Input:  
num = 55911  

Output:  
2  

**Explanation:**  
The digits `5` and `1` are repeating.

### Example 2
Input:  
num = 2491  

Output:  
0  

**Explanation:**  
No digits are repeating in the number.

### Your Task
Complete the function `countRepeatingDigits()` which takes an integer `num` as an argument and returns the count of digits that appear more than once in `num`.

### Constraints
- 0 ≤ num ≤ 10⁹  

### Input Format
- Input is a single integer `num`.

### Output Format
- Output is a single integer representing the count of repeating digits in `num`.
