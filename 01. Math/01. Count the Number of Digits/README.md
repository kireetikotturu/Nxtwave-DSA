# Count the Number of Digits

**Difficulty:** Easy  
**Time Complexity:** O(log₁₀N)  
**Space Complexity:** O(1)  

---

You are given an integer `N`.  
Your task is to calculate and return the number of digits of the given integer `N`.

### Example 1
Input:  
N = 123  

Output:  
3  

**Explanation:**  
The number `123` has 3 digits.

### Example 2
Input:  
N = 4562  

Output:  
4  

**Explanation:**  
The number `4562` has 4 digits.

### Your Task
Implement the function `countDigits()` that takes an integer `N` as its argument and returns the number of digits in `N`.

### Constraints
- 1 ≤ N ≤ 10⁹  

### Input Format
- The input consists of a single integer `N`.

### Output Format
- Single integer, representing the count of digits in `N`.
