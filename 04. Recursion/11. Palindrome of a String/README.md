# Palindrome of a String

**Difficulty:** Easy  
**Time Complexity:** O(N)  
**Space Complexity:** O(N)  
**Companies:** Uber, Apple, Amazon, Zoho, Google, Zenefits, Microsoft, Bloomberg  

---

You are given a string `s`.  
Your task is to check whether the given string is a **palindrome** or not.

### Note
A palindrome is a sequence of characters that reads the same **forward and backward**.

Examples of palindromes:
- `level`
- `radar`
- `racecar`

Non-palindromes:
- `hello`
- `textbox`

### Example 1
Input:  
s = "racecar"  

Output:  
true  

**Explanation:**  
The string reads the same forward and backward.

### Example 2
Input:  
s = "hello"  

Output:  
false  

**Explanation:**  
The string does not read the same backward as forward.

### Your Task
Complete the function `checkPalindrome()` that takes a string `s` as input and returns `true` if it is a palindrome, otherwise `false`.

### Constraints
- 0 ≤ s.length ≤ 100  

### Input Format
- A single line containing a string `s`.

### Output Format
- Output `true` if the string is a palindrome; otherwise, output `false`.
