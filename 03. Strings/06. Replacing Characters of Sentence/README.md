# Replacing Characters of Sentence

**Difficulty:** Medium  
**Time Complexity:** O(N)  
**Space Complexity:** O(N)  

---

You are given a string `S`.  
Your task is to **replace each letter with the next letter in the English alphabet**.

### Rules
- Uppercase letters must be replaced with **uppercase**
- Lowercase letters must be replaced with **lowercase**
- Spaces should remain unchanged
- `'z'` should be replaced with `'a'`
- `'Z'` should be replaced with `'A'`

### Example 1
Input:  
Hello World

Output:  
Ifmmp Xpsme

**Explanation:**  
Each character is replaced by the next alphabet letter while preserving case.

### Example 2
Input:  
Something is better than Nothing

Output:  
Tpnfuijoh jt cfuufs uibo Opuijoh

### Your Task
Read the input string and print the transformed string after replacing each character according to the rules.

### Input Format
- A single line containing the string `S`

### Output Format
- A single line containing the modified string
